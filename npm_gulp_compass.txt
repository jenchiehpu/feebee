沒有gulp的時期
編譯scss或是sass成css時，需要compass程式，但每次存檔後編譯手動編譯很麻煩，因此有人會寫script自動化完成，但不只scss轉css，還需要minify javascript或是css的代碼，又要寫一個script去啟動uglify這類的程式，因此gulp這種會自動監聽，比對(watch)的程式就很方便囉!


sass是基於ruby開發的程式，不只是副檔名.sass或是.scss，也是一個程式，能編譯scss或是sass的檔案成css的語法，使瀏覽器能看懂。
因此先安裝ruby，然後安裝sass，就可以編譯scss檔案。編譯方式都是要打指令，有的人覺得每次打指令很麻煩，因此下達watch指令，每次存檔sass就會自編譯

$ sudo apt install ruby
$ ruby -v
$ sudo gem install sass
以上三行就是安裝指令

$ sass A.scss A.css
編譯的指令

$ sass --watch A.scss:A.css
自動監聽檔案

$ sass --watch A folder:B folder
自動監聽資料夾
